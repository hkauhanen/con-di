Henri Kauhanen 2020

1 Notation

The following notational conventions are adopted (cf. Appendix B of the paper): x stands for x1, y stands for x2, f stands for f1, and g stands for f2. tx means tilde'ed x, and so on.

2 Fitnesses

Here, we define the fitnesses that determine the game in the fully symmetric case.

(%i1) tx : 1 − x ;
(tx) 1 − x
(%i2) ty : 1 − y ;
(ty) 1 − y
(%i3) f : x + σ· x· ty ;
(f) x ⁢ ( 1 − y ) ⁢ σ + x
(%i4) g : y + σ· y· tx ;
(g) ( 1 − x ) ⁢ y ⁢ σ + y
(%i5) tf : tx + σ· tx· y ;
(tf) ( 1 − x ) ⁢ y ⁢ σ − x + 1
(%i6) tg : ty + σ· ty· x ;
(tg) x ⁢ ( 1 − y ) ⁢ σ − y + 1

3 Time derivative and Jacobian

Define time derivative and form Jacobian matrix.

(%i7) dotx : ( tx − μ)· x· f − ( x − μ)· tx· tf ;
(dotx) x ⁢ ( − μ − x + 1 ) ⁢ ( x ⁢ ( 1 − y ) ⁢ σ + x ) − ( 1 − x ) ⁢ ( x − μ ) ⁢ ( ( 1 − x ) ⁢ y ⁢ σ − x + 1 )
(%i8) doty : ( ty − μ)· y· g − ( y − μ)· ty· tg ;
(doty) y ⁢ ( − μ − y + 1 ) ⁢ ( ( 1 − x ) ⁢ y ⁢ σ + y ) − ( 1 − y ) ⁢ ( y − μ ) ⁢ ( x ⁢ ( 1 − y ) ⁢ σ − y + 1 )
(%i9) der : [ dotx, doty] ;
(der) [ x ⁢ ( − μ − x + 1 ) ⁢ ( x ⁢ ( 1 − y ) ⁢ σ + x ) − ( 1 − x ) ⁢ ( x − μ ) ⁢ ( ( 1 − x ) ⁢ y ⁢ σ − x + 1 ) , y ⁢ ( − μ − y + 1 ) ⁢ ( ( 1 − x ) ⁢ y ⁢ σ + y ) − ( 1 − y ) ⁢ ( y − μ ) ⁢ ( x ⁢ ( 1 − y ) ⁢ σ − y + 1 ) ]
(%i10) J : jacobian( der, [ x, y]) ;
(J) ( ( x − μ ) ⁢ ( ( 1 − x ) ⁢ y ⁢ σ − x + 1 ) − ( 1 − x ) ⁢ ( ( 1 − x ) ⁢ y ⁢ σ − x + 1 ) − ( 1 − x ) ⁢ ( x − μ ) ⁢ ( − y ⁢ σ − 1 ) + ( − μ − x + 1 ) ⁢ ( x ⁢ ( 1 − y ) ⁢ σ + x ) − x ⁢ ( x ⁢ ( 1 − y ) ⁢ σ + x ) + x ⁢ ( − μ − x + 1 ) ⁢ ( ( 1 − y ) ⁢ σ + 1 ) − ( 1 − x ) 2 ⁢ ( x − μ ) ⁢ σ − x 2 ⁢ ( − μ − x + 1 ) ⁢ σ − ( 1 − y ) 2 ⁢ ( y − μ ) ⁢ σ − y 2 ⁢ ( − μ − y + 1 ) ⁢ σ ( − μ − y + 1 ) ⁢ ( ( 1 − x ) ⁢ y ⁢ σ + y ) − y ⁢ ( ( 1 − x ) ⁢ y ⁢ σ + y ) + ( y − μ ) ⁢ ( x ⁢ ( 1 − y ) ⁢ σ − y + 1 ) − ( 1 − y ) ⁢ ( x ⁢ ( 1 − y ) ⁢ σ − y + 1 ) − ( 1 − y ) ⁢ ( y − μ ) ⁢ ( − x ⁢ σ − 1 ) + y ⁢ ( − μ − y + 1 ) ⁢ ( ( 1 − x ) ⁢ σ + 1 ) )

4 Zero-mutation case

If μ = 0, there are nine equilibria.

(%i11) der0 : [ subst([ μ = 0], dotx), subst([ μ = 0], doty)] ;
(der0) [ ( 1 − x ) ⁢ x ⁢ ( x ⁢ ( 1 − y ) ⁢ σ + x ) − ( 1 − x ) ⁢ x ⁢ ( ( 1 − x ) ⁢ y ⁢ σ − x + 1 ) , ( 1 − y ) ⁢ y ⁢ ( ( 1 − x ) ⁢ y ⁢ σ + y ) − ( 1 − y ) ⁢ y ⁢ ( x ⁢ ( 1 − y ) ⁢ σ − y + 1 ) ]
(%i12) sol0 : solve( der0, [ x, y]) ;
(sol0) [ [ x = 0 , y = 0 ] , [ x = 1 σ + 2 , y = 0 ] , [ x = 1 , y = 0 ] , [ x = 0 , y = 1 σ + 2 ] , [ x = 1 2 , y = 1 2 ] , [ x = 1 , y = σ + 1 σ + 2 ] , [ x = 0 , y = 1 ] , [ x = σ + 1 σ + 2 , y = 1 ] , [ x = 1 , y = 1 ] ]

5 Stability of uniform state

Reversals of stability of the uniform state u = (1/2, 1/2) are all important. Here, we get the eigenvalues of the Jacobian, evaluated at u.

(%i23) u : [ 1 / 2, 1 / 2] ;
(u) [ 1 2 , 1 2 ]
(%i24) u_eig : eigenvalues( subst([ x = u[ 1], y = u[ 2]], J)) ;
(u_eig) [ [ − μ ⁢ ( 3 ⁢ σ + 4 ) − σ − 1 2 , − μ ⁢ ( σ + 4 ) − 1 2 ] , [ 1 , 1 ] ]

These have zeroes at the following values of μ. These are two of our critical values of the mutation parameter.

(%i25) solve( u_eig[ 1][ 1], μ) ;
(%o25) [ μ = σ + 1 3 ⁢ σ + 4 ]
(%i26) solve( u_eig[ 1][ 2], μ) ;
(%o26) [ μ = 1 σ + 4 ]

In the full-mutation (μ = 1) case, we have the following in particular.

(%i27) u_eig1 : subst([ μ = 1], u_eig) ;
(u_eig1) [ [ − 2 ⁢ σ + 3 2 , − σ + 3 2 ] , [ 1 , 1 ] ]

6 Diagonal equilibria

For some values of μ, the game has equilibria on the diagonals D (where x = y) and tD (where x = 1 – y) different from the uniform state u = (1/2, 1/2). We solve for these by carrying out the relevant substitutions. Note that, due to symmetry, it suffices to consider dotx only.

(%i28) dotx_tD : subst( y = 1 − x, dotx) ;
(dotx_tD) x ⁢ ( − μ − x + 1 ) ⁢ ( x 2 ⁢ σ + x ) − ( 1 − x ) ⁢ ( x − μ ) ⁢ ( ( 1 − x ) 2 ⁢ σ − x + 1 )
(%i29) sols_tD : solve( dotx_tD, x) ;
(sols_tD) [ x = − ( − 3 ⁢ μ 2 − 2 ⁢ μ + 1 ) ⁢ σ 2 + ( − 4 ⁢ μ 2 − 6 ⁢ μ + 2 ) ⁢ σ − 4 ⁢ μ + 1 + ( − μ − 1 ) ⁢ σ − 1 ( 2 ⁢ μ + 2 ) ⁢ σ + 2 , x = ( − 3 ⁢ μ 2 − 2 ⁢ μ + 1 ) ⁢ σ 2 + ( − 4 ⁢ μ 2 − 6 ⁢ μ + 2 ) ⁢ σ − 4 ⁢ μ + 1 + ( μ + 1 ) ⁢ σ + 1 ( 2 ⁢ μ + 2 ) ⁢ σ + 2 , x = 1 2 ]
(%i30) dotx_D : subst( y = x, dotx) ;
(dotx_D) x ⁢ ( − μ − x + 1 ) ⁢ ( ( 1 − x ) ⁢ x ⁢ σ + x ) − ( 1 − x ) ⁢ ( x − μ ) ⁢ ( ( 1 − x ) ⁢ x ⁢ σ − x + 1 )
(%i31) sols_D : solve( dotx_D, x) ;
(sols_D) [ x = − μ 2 ⁢ σ 2 + ( 4 ⁢ μ 2 − 2 ⁢ μ ) ⁢ σ − 4 ⁢ μ + 1 − μ ⁢ σ + 1 2 ⁢ μ ⁢ σ − 2 , x = μ 2 ⁢ σ 2 + ( 4 ⁢ μ 2 − 2 ⁢ μ ) ⁢ σ − 4 ⁢ μ + 1 + μ ⁢ σ − 1 2 ⁢ μ ⁢ σ − 2 , x = 1 2 ]

7 Stability of D-diagonal equilibria

The D-diagonal equilibria both bifurcate to yield further equilibria as μ approaches 0. Although it has not been possible to analytically solve for those new equilibria (which lie outside each diagonal), we can identify the bifurcation points by examining the zero-crossing behaviour of the eigenvalues of the Jacobian evaluated at the D-diagonal equilibria.

(%i32) D_eq : [ sols_D[ 1], subst([ x = y], sols_D[ 1])] ;
(D_eq) [ x = − μ 2 ⁢ σ 2 + ( 4 ⁢ μ 2 − 2 ⁢ μ ) ⁢ σ − 4 ⁢ μ + 1 − μ ⁢ σ + 1 2 ⁢ μ ⁢ σ − 2 , y = − μ 2 ⁢ σ 2 + ( 4 ⁢ μ 2 − 2 ⁢ μ ) ⁢ σ − 4 ⁢ μ + 1 − μ ⁢ σ + 1 2 ⁢ μ ⁢ σ − 2 ]
(%i33) eig_D : eigenvalues( subst( D_eq, J)) ;
(eig_D) [ [ − μ 2 ⁢ σ 2 + μ ⁢ ( 2 ⁢ σ + 4 ) − 1 μ ⁢ σ − 1 , μ ⁢ ( σ + 4 ) − 1 ] , [ 1 , 1 ] ]

Zeroes. The second element of the first of these is our third critical value of the mutation parameter.

(%i34) solve( eig_D[ 1][ 1], μ) ;
(%o34) [ μ = − 2 ⁢ σ 2 + 2 ⁢ σ + 2 + σ + 2 σ 2 , μ = 2 ⁢ σ 2 + 2 ⁢ σ + 2 − σ − 2 σ 2 ]
(%i35) solve( eig_D[ 1][ 2], μ) ;
(%o35) [ μ = 1 σ + 4 ]
